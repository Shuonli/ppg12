# A New Critical Study of Photon Production in Hadronic Collisions

## Abstract

In the light of the new prompt photon data collected by PHENIX at RHIC and by D0 at the run II of the Tevatron, the world prompt photon data, both inclusive and isolated, in hadronic collisions, are revisited and compared with the NLO QCD calculations implemented in the Monte Carlo programme JETPHOX. Agreement between data and NLO QCD is found across nine orders of magnitude in cross section, from sqrt(s) = 23 GeV to 1.96 TeV, with the notable exception of the E706 fixed-target experiment data.

## Key Results

- JETPHOX is a general-purpose NLO partonic Monte Carlo for prompt photon production in hadronic collisions, computing both the direct (D) and fragmentation (F) components at NLO accuracy in alpha_s.
- Excellent agreement between NLO predictions and world data across nine orders of magnitude in the cross section, from sqrt(s) = 23 GeV (WA70, UA6) to 1.96 TeV (D0 Run II).
- The E706 data remain the sole exception, requiring a non-perturbative "kT kick" that no other experiment needs.
- PHENIX data at sqrt(s) = 200 GeV (both inclusive and isolated) are well described by JETPHOX with scale mu = pT/2.
- Scale sensitivity is about +/-10% when varying the common scale from pT/2 to 2pT.
- Changing PDF from CTEQ6M to MRST 2004 shifts predictions by only +/-2%.
- The fragmentation contribution (F) is dominant at low pT at collider energies but is strongly suppressed by isolation cuts (photons with z >= (1 + epsilon_h)^{-1} survive).

## Methodology / Formalism

### Cross-section decomposition

The prompt photon differential cross section is written as:

    d sigma^gamma = d sigma^(D)(mu_R, M, M_F) + sum_k sigma^(F)_k(mu_R, M, M_F) x D^{gamma/k}(M_F)

where:
- sigma^(D): direct component (Born + higher-order corrections with final-state collinear singularities subtracted in MS scheme)
- sigma^(F): fragmentation component (involves fragmentation functions D^{gamma/k})
- mu_R: renormalization scale
- M: initial-state factorization scale
- M_F: fragmentation scale

### Direct component at NLO

    d sigma^(D) / d pT d eta = sum_{i,j} int dx1 dx2 F_{i/h1}(x1, M) F_{j/h2}(x2, M)
        x (alpha_s/(2 pi)) [ d hat{sigma}_{ij} / d pT d eta + (alpha_s/(2 pi)) K^(D)_{ij}(mu_R, M, M_F) ]

### Fragmentation component at NLO

    d sigma^(F) / d pT d eta = sum_{i,j,k} int dx1 dx2 (dz/z^2) F_{i/h1}(x1, M) F_{j/h2}(x2, M) D^{gamma/k}(z, M_F)
        x (alpha_s/(2 pi))^2 [ d hat{sigma}^k_{ij} / d pT d eta + (alpha_s/(2 pi)) K^(F)_{ij,k}(mu_R, M, M_F) ]

### JETPHOX implementation

- Combined phase-space slicing and subtraction method for IR singularities
- Phase space partitioned using unphysical parameters pTm and RTh into four parts (Part I: soft; Parts IIa, IIb: final-state collinear; Part IIc: finite remainder)
- Collinear and soft singularities regularized by dimensional continuation (n = 4 - 2 epsilon)
- After combining with virtual contributions, IR singularities cancel; remaining collinear singularities absorbed into PDFs and FFs
- MS factorization scheme used throughout

### Scale choices

- Standard: mu_R = M = M_F = mu (common scale)
- Preferred: mu = pT/2 (motivated by NLL resummation calculations; best fit to fixed-target and ISR data)
- Scale variation: pT/2 to 2pT provides ~10% uncertainty envelope
- Separate scale variation also studied: e.g., mu_R = pT, M = 2pT, M_F = pT/2

### Isolation criterion

- Cone criterion: hadronic E_T inside cone R = sqrt(Delta phi^2 + Delta eta^2) required to be less than E_T^max
- E_T^max given either as fixed value or as fixed fraction epsilon_h of photon pT
- Isolation reduces fragmentation contribution but does not eliminate it (surviving fraction at z >= (1 + epsilon_h)^{-1})
- NLO calculation may become unreliable for R <= 0.3 due to unresummed large log(R) terms

### Inputs used

- PDFs: CTEQ6M (alpha_s(M_Z) = 0.118)
- Fragmentation functions: BFG set II (Bourhis, Fontannaz, Guillet)
- MS renormalization scheme

## Relevance to PPG12

- **JETPHOX is the primary NLO theory program** used in PPG12 for computing the theoretical isolated photon cross section at sqrt(s) = 200 GeV. The PPG12 pipeline runs JETPHOX at three scales (mu = 0.5 ET, ET, 2 ET) to produce the theory band shown in the final cross-section plot.
- The JETPHOX ROOT output is processed by `NLO/MakeJetPHOXhisto.C` which reads pawres output trees and bins into the PPG12 pT bins.
- The PPG12 final plot (`plotting/plot_final_selection.C`) overlays data with the JETPHOX NLO band (labeled "NLO pQCD JETPHOX") and annotates "CT14 PDF / BFG II FF".
- The paper validates that PHENIX isolated photon data at sqrt(s) = 200 GeV agree with JETPHOX using mu = pT/2, confirming that this same code is reliable at sPHENIX kinematics.
- The JETPHOX installation used by PPG12 is at `/sphenix/user/shuhangli/jetphox/newbuild/jetphox_1.3.1_4/`.
