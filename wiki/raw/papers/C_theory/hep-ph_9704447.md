# Quark and Gluon Fragmentation Functions into Photons

## Abstract

The fragmentation functions of quarks and gluons into photons are studied beyond the Leading Logarithm approximation. The nature of the initial conditions of the evolution equation solutions is addressed, and problems related to factorization scheme invariance are studied. The possibility of measuring these distributions in LEP experiments is discussed, and a comparison with existing data is made.

## Key Results

- Provides the BFG (Bourhis-Fontannaz-Guillet) parameterization of parton-to-photon fragmentation functions D^{gamma}_q(z, M^2) and D^{gamma}_g(z, M^2) at Beyond Leading Logarithm (BLL) accuracy.
- Two sets produced: **BFG set I** ("small gluon") and **BFG set II** ("large gluon"), differing in the non-perturbative gluon-to-photon fragmentation input constrained by rho-meson production data.
- The full fragmentation function is decomposed as D^{gamma} = D^{gamma,AN} (anomalous/perturbative) + D^{gamma,NP} (non-perturbative), where the anomalous part is calculable in pQCD and the non-perturbative part is modeled via Vector Meson Dominance (VDM).
- The non-perturbative initial condition at M^2 = M_0^2 ~ m_rho^2 ~ 0.5 GeV^2 is scheme-dependent and includes a perturbative correction term D^{gamma,MS}(z) beyond the VDM contribution.
- At BLL accuracy, the quark fragmentation function at z -> 1 behaves as D^{gamma}_q ~ (alpha/(2 pi)) ln(1/(1-z)), which is cancelled by the direct coefficient function in physical observables.
- The gluon fragmentation function D^{gamma}_g(z, M^2) becomes negative at small z due to BLL kernel singularities, but this region (z < 0.01) is not probed experimentally.

## Methodology / Formalism

### DGLAP evolution equations

The fragmentation functions satisfy inhomogeneous DGLAP equations. In the non-singlet sector:

    M^2 dD^{gamma}_{ns,i} / dM^2 = C_{ns,i} K^{gamma}_q + P_{ns} x D^{gamma}_{ns,i}

In the singlet sector:

    M^2 dD^{gamma}_q / dM^2 = C_s K^{gamma}_q + P_{qq} x D^{gamma}_q + P_{gq} x D^{gamma}_g
    M^2 dD^{gamma}_g / dM^2 = C_s K^{gamma}_g + P_{qg} x D^{gamma}_q + P_{gg} x D^{gamma}_g

where the inhomogeneous kernels K^{gamma}_a have perturbative expansion:

    K^{gamma}_a(z, M^2) = (alpha / 2 pi) [ K^(0)_{gamma a}(z) + (alpha_s / 2 pi)(M^2) K^(1)_{gamma a}(z) + ... ]

The gluon coupling to the photon proceeds only through a quark loop, so K^{gamma}_g starts at O(alpha_s).

### Decomposition of the full solution

    D^{gamma}(n, M^2) = D^{gamma,AN}(n, M^2) + D^{gamma,NP}(n, M^2)

- **Anomalous (AN)**: fully calculable in pQCD; at LL behaves as ~ (4 pi / alpha_s) ln(M^2 / Lambda^2)
- **Non-perturbative (NP)**: initial condition at M_0^2, modeled via VDM; satisfies homogeneous DGLAP evolution for M^2 > M_0^2

### Non-perturbative initial condition (MS scheme)

    D^{gamma,NP}_{ns,i}(z, M_0^2) = D^{gamma,VDM}_{ns,i}(z, M_0^2) - C_{ns,i} D^{gamma,MS}(z)

where the perturbative correction is:

    D^{gamma,MS}(z) = (alpha / 2 pi) [ (1 + (1-z)^2) / z ] [ ln(1-z) + ln(z) ] + z ]

This ensures factorization scheme invariance of the physical cross section.

### Heavy quark treatment

- Heavy quark threshold at mu^2 = m_c^2 and mu^2 = m_b^2
- D^{gamma}_{c}(mu^2 < m_c^2) = 0 and D^{gamma}_{b}(mu^2 < m_b^2) = 0
- MS scheme input at the mass threshold:
    D^{gamma,MS}_Q(z, m_Q^2) = -(alpha / 2 pi) e_Q^2 [ (1 + (1-z)^2) / z ] (2 ln z + 1)
- Parameters: Lambda^(4)_QCD = 230 MeV, m_charm = 1.5 GeV, m_bottom = 4.5 GeV

### VDM model for non-perturbative input

The photon is represented as a superposition of vector mesons (rho, omega, phi) via VDM:

    gamma = g sqrt(2) [ rho + omega/3 - sqrt(2)/3 phi ]

The non-perturbative fragmentation is modeled from rho^0-meson fragmentation data:

    D^{gamma,VDM}_q = (g^2) [ (4/9) D^{(u ubar)}_q + (1/9) D^{(d dbar)}_q + (1/9) D^{(s sbar)}_q ]

with g^2 ~ alpha.

### VDM parametrization (at Q_0^2 = 2 GeV^2)

Fragmentation functions into rho parametrized as D = N x^alpha (1-x)^beta with:

| Set   | Valence (N, alpha, beta) | Sea (N, alpha, beta) | Gluon (N, alpha, beta) |
|-------|-------------------------|---------------------|----------------------|
| Set I | (0.785, -0.5, 1.499)   | (0.111, -1, 2.912)  | (0.108, -1, 3)       |
| Set II| (1.140, -0.2, 1.693)   | (0.100, -0.3, 3.437)| (2.550, -0.3, 3)     |

- Set I: chi^2/dof = 1.33, constrained with fixed beta_g = 3
- Set II: chi^2/dof = 1.22, gluon normalization increased to match D^{rho}_g / D^{rho}_u ratio comparable to D^{pi0}_g / D^{pi0}_u
- Data from ALEPH and HRS used for fits

## Relevance to PPG12

- **BFG set II is used in JETPHOX** for the PPG12 NLO theory comparison. The PPG12 final plot legend explicitly states "CT14 PDF / BFG II FF".
- The choice of BFG II vs BFG I has negligible impact on isolated photon spectra because isolation cuts suppress ~60-80% of fragmentation photons, and the sets differ mainly in the gluon FF at low z.
- The parton-to-photon fragmentation functions enter the fragmentation component of the cross section, which at sqrt(s) = 200 GeV and with PPG12 isolation (R = 0.3, parametric cut) contributes at the 10-20% level.
- Understanding the BFG parameterization is important for systematic uncertainty estimation: the JETPHOX scale variation (mu = 0.5 ET to 2 ET) implicitly includes some sensitivity to the fragmentation scale M_F.
