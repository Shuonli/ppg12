# Isolated Photons in Perturbative QCD

## Abstract

A definition of the cross section for the production of an isolated photon plus n jets is presented which only depends upon direct photon production, and is independent of the parton-to-photon fragmentation contribution. This prescription, based on a modified cone approach which implements the isolation condition in a smooth way, treats quarks and gluons identically and can be directly applied to experimental data in hadron-hadron, photon-hadron and e+e- collisions. The case of several isolated photons in the final state can also be dealt with in the very same way.

## Key Results

- **Frixione smooth cone isolation** completely eliminates the fragmentation contribution to the isolated photon cross section, making it depend only on the direct production mechanism.
- The prescription is infrared (IR) safe to all orders in perturbative QCD: soft gluons can be arbitrarily close to the photon (satisfying IR cancellation), while collinear partons are forced to have vanishing energy (suppressing fragmentation).
- The fragmentation contribution is restricted to the zero-measure set z = 1 in momentum fraction, making it vanish.
- For standard fixed-cone isolation (epsilon_h, R), there is a fundamental tension: non-zero E_T^max preserves IR safety but allows some fragmentation; zero E_T^max kills fragmentation but spoils IR cancellation. The Frixione prescription resolves this by smoothly interpolating.
- The prescription introduces moderate negative NLO corrections compared to standard fixed-cone isolation, controlled by the parameter n in the threshold function.

## Methodology / Formalism

### The smooth isolation condition

For each event, a cone of half-angle (or radius) delta_0 is drawn around the photon. For **all** delta <= delta_0, the total hadronic transverse energy inside the sub-cone of radius delta must satisfy:

    sum_i E_T^i theta(delta - R_{i gamma}) <= X(delta)   for all delta <= delta_0

where R_{i gamma} = sqrt((eta_i - eta_gamma)^2 + (phi_i - phi_gamma)^2) is the angular distance in the eta-phi plane.

### The threshold function X(delta)

    X(delta) = E_T^gamma epsilon_gamma [ (1 - cos delta) / (1 - cos delta_0) ]^n

with the canonical choice:

    epsilon_gamma = 1,   n = 1

The key property: X(delta) -> 0 as delta -> 0.

### Why it works: IR safety + fragmentation suppression

**Soft gluons**: For a soft parton (E -> 0), the isolation cone provides no constraint (Eq. 3.10 vanishes), so soft gluon cancellation proceeds as in standard jet cross sections.

**Collinear quarks**: When a quark approaches the photon direction (delta -> 0), the isolation condition forces E_quark -> 0 proportionally to delta^{2n}. This energy damping cancels the 1/(1-y) collinear divergence. Explicitly, the contribution from inside the isolation cone:

    sigma_cone ~ E_gamma^2 epsilon_gamma^2 / (4n)

which is finite for n >= 1/2.

**Collinear gluons**: The finite contribution from a gluon inside the cone:

    sigma_cone ~ (1 - cos delta_0) [ log(E_gamma epsilon_gamma) - n ]

For the standard fixed-cone approach (n = 0, epsilon_gamma = epsilon_c), this gives a large negative correction log(epsilon_c). With n = 1, the corrections are moderate since epsilon_gamma need not be small.

### Comparison with standard cone isolation

Standard cone isolation corresponds to n = 0, epsilon_gamma = epsilon_c (small). The Frixione prescription (n = 1, epsilon_gamma = 1):
- Eliminates the large negative correction ~ log(epsilon_c) present in standard isolation
- Allows a larger cone delta_0 without producing large log(1 - cos delta_0) terms
- Completely removes fragmentation function dependence

### Infrared safety proof (argument)

Given a configuration that fulfills the isolation criteria, a parton emitting a soft gluon or splitting into collinear partons does not abruptly change the physical observables. Conversely, a non-isolated configuration remains non-isolated after soft emission or collinear splitting. Therefore QCD infrared configurations can be locally subtracted, and the corresponding singularities are cancelled by virtual contributions. The only remaining singularities are QED (quarks collinear to the photon), which are damped by the energy suppression mechanism.

### Experimental practicalities

When delta ~ 0, the threshold X(delta) falls below the calorimeter energy threshold E_th, making the condition experimentally irrelevant in that limit. Since E_th ~ O(100 MeV) and the photon is hard, the contribution from fragmentation at z > z_th ~ 1 - E_th/E_gamma is negligibly small.

## Relevance to PPG12

- **Conceptual foundation**: The Frixione isolation defines the theoretical gold standard for isolated photon measurements. While PPG12 uses a standard cone isolation (R = 0.3, parametric E_T cut), understanding the Frixione prescription is essential for interpreting the residual fragmentation contribution and the associated theoretical uncertainty.
- **JETPHOX can run with either isolation**: PPG12's JETPHOX runs use the standard cone criterion matching the experimental selection, which retains a small fragmentation contribution. The Frixione isolation would eliminate this but does not match the experimental procedure.
- The distinction is quantitatively small for the PPG12 kinematic range (high pT, tight isolation), as shown by Ichou & d'Enterria (1005.4529): standard and smooth-cone isolation predictions agree within ~5% at high ET.
- The Catani et al. paper (hep-ph/0204023) provides the NLO implementation of standard cone isolation in JETPHOX, proving factorization for the non-smooth case.
