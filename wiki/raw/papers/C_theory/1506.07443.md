# New Parton Distribution Functions from a Global Analysis of Quantum Chromodynamics

## Abstract

New parton distribution functions (PDFs) at next-to-next-to-leading order (NNLO) from the CTEQ-TEA global analysis of quantum chromodynamics are presented. These differ from previous CT PDFs in several respects, including the use of data from LHC experiments, and the new D0 charged lepton rapidity asymmetry data, as well as the use of a more flexible parametrization of PDFs that allows a better fit to different combinations of quark flavors. Predictions for important LHC processes, especially Higgs boson production at 13 TeV, are presented. The CT14 PDFs include a central set and error sets in the Hessian representation. CT14 PDFs are also provided at leading order (LO) and next-to-leading order (NLO) in QCD. Additional sets with various alpha_s values and general-mass variable flavor number (GM-VFN) schemes with up to 3, 4, and 6 active flavors are provided.

## Key Results

- **CT14 NNLO**: 28 free parameters (vs 25 in CT10), fitted to 2947 data points from 33 experiments, yielding chi^2/N_pt = 1.10.
- **PDF sets provided**: NNLO (central + 56 Hessian error sets), NLO, and two LO sets (one with 1-loop alpha_s(M_Z) = 0.130, one with 2-loop alpha_s(M_Z) = 0.118).
- **alpha_s(M_Z)**: Central value 0.118 used for NLO and NNLO fits. CT14 prefers alpha_s(M_Z) = 0.115 (+0.006/-0.004) at NNLO, 0.117 +/- 0.005 at NLO (90% C.L.).
- **New parametrization**: Bernstein polynomials replace the previous polynomial form, reducing undesirable correlations and allowing arbitrary constant ratios of d/u and dbar/ubar as x -> 0 or x -> 1.
- **Gluon PDF**: Increased by 1-2% at x ~ 0.05 compared to CT10, somewhat modified at x > 0.1 by LHC jet data and multiplicative treatment of correlated errors. The gluon is primarily constrained by inclusive jet production (Tevatron + LHC) and DIS scaling violations.
- **Strangeness**: Softer s(x, Q) at low x compared to CT10 NNLO.
- **d/u ratio**: Decreased at high x due to updated D0 electron charge asymmetry data (9.7 fb^-1 replacing 0.75 fb^-1); approaches a constant value at x -> 1.
- **LHC data included**: ATLAS and CMS W/Z cross sections (7 TeV), CMS electron/muon charge asymmetry, LHCb W/Z rapidity distributions, ATLAS and CMS inclusive jet production.
- **Higgs production**: sigma(gg -> H) at 13 TeV = 42.7 pb (+ 3.0% / -3.2% PDF+alpha_s), increased 1.1% compared to CT10 NNLO.

## Methodology / Formalism

### Global fit framework

- **Figure of merit**: chi^2_global = sum_n chi^2_n + chi^2_th, where chi^2_n accounts for uncorrelated and correlated experimental errors per data set, and chi^2_th enforces theoretical constraints (Lagrange multiplier).
- **Error sets**: Hessian method with iterative diagonalization. 90% C.L. boundary determined by Delta chi^2 = T^2 with T ~ 10, plus Tier-2 penalty P = sum_n (S_n)^k theta(S_n) with k = 16 and S_n the effective Gaussian variable.
- **Monte Carlo sampling**: Also used for verification, especially in regions not well constrained by data.
- **S_n variable**: Maps chi^2_n values (whose probability distributions depend on N_pt,n) onto a universal standard normal distribution. Values between -1 and +1 indicate good fit (68% C.L.).

### Parametrization

PDFs parametrized at initial scale Q_0 = 1.295 GeV using Bernstein polynomials:

    x f(x, Q_0) = x^{a_1} (1-x)^{a_2} sum_i c_i B_i(x)

where B_i(x) are Bernstein basis functions. Properties:
- Single polynomial dominant in each x range, reducing parameter correlations
- Asymptotic power x^{a_1} shared by all light-quark PDFs at x -> 0
- dval and uval share the same (1-x)^{a_2} exponent at x -> 1 (spectator counting rules)
- Independent normalizations allowed for d/u at x -> 1

### Heavy quark treatment

- Charm pole mass: m_c = 1.3 GeV (consistent with CT10 data)
- Charm PDF turned on at Q = m_c with zero intrinsic charm
- S-ACOT-chi factorization scheme at NNLO for massive quarks in neutral-current DIS
- s(x, Q_0) = sbar(x, Q_0) assumed (consistent with data)

### Theoretical calculations

- Neutral-current DIS: NNLO in S-ACOT-chi scheme
- Drell-Yan: NNLO via VRAP
- W/Z with pT cuts: NNLL (approx. NNLO) via ResBos
- Charged-current DIS: NLO only (NNLO not yet complete)
- Inclusive jets: NLO only, via FastNLO/ApplGrid interface to NLOJET++, with mu_R = mu_F = pT of individual jet
- Correlated systematic errors: multiplicative treatment for all experiments

### Correlation studies

- Jet cross sections are highly correlated with g(x, Q) at x > 0.05 (Tevatron) and x > 0.005 (LHC)
- CMS charge asymmetry is moderately correlated with dbar/ubar and d/u at x ~ 0.01
- HERA data constrain gluon at x < 0.01 via scaling violations

## Relevance to PPG12

- **CT14 LO is the PDF set used in JETPHOX** for the PPG12 NLO theory comparison. The final plot legend in `plot_final_selection.C` reads "CT14 PDF / BFG II FF".
- **Note on LO vs NLO vs NNLO**: JETPHOX is an NLO calculation, so using CT14 NLO PDFs would be formally consistent. The use of CT14lo (with 2-loop alpha_s(M_Z) = 0.118) in the PPG12 pipeline should be verified against the JETPHOX configuration. CT14 provides both LO sets: one with 1-loop alpha_s = 0.130 (for LO matrix element generators like PYTHIA) and one with 2-loop alpha_s = 0.118 (for NLO matching).
- **Gluon PDF uncertainty**: The gluon PDF at the x values probed by sqrt(s) = 200 GeV isolated photon production (x ~ 0.04-0.35 for pT = 8-35 GeV at midrapidity) has a ~3-5% uncertainty at Q = 100 GeV. This contributes to the theoretical uncertainty band on the JETPHOX prediction.
- **PDF error sets**: The 56 Hessian error sets could in principle be used to compute a separate PDF uncertainty band on the JETPHOX prediction, though the PPG12 analysis currently uses only the central set with scale variation as the theory band.
- **alpha_s sensitivity**: The JETPHOX runs use alpha_s(M_Z) = 0.118, matching the CT14 central value.
