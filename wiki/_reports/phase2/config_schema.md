# PPG12 YAML Configuration Schema Reference

## Overview

The PPG12 analysis uses two distinct YAML configuration schemas:

1. **Training config** (`FunWithxgboost/config.yaml`) -- controls XGBoost BDT training pipeline (Python)
2. **Analysis config** (`FunWithxgboost/config_nom.yaml`, `efficiencytool/config_bdt_*.yaml`, `efficiencytool/config_showershape_*.yaml`) -- controls efficiency calculation, ABCD analysis, shower shape studies, and BDT application (C++)
3. **NPB training config** (`FunWithxgboost/config_npb_training.yaml`) -- controls NPB score BDT training (Python, separate pipeline)

There are 37 `config_bdt_*.yaml` files in `efficiencytool/` (generated by `make_bdt_variations.py` from a nominal base) plus multiple `config_showershape_*.yaml` files for shower shape studies. All analysis configs share the same schema.
