## How to use this list

- **★★★** = cite in analysis note / ingest into wiki immediately
- **★★** = cite if relevant section is written / ingest for domain context
- **★** = optional reading / ingest if time permits
- **[ALREADY CITED]** = already in bib files; included inline for completeness as wiki ingest candidates
